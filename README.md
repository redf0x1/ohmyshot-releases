## Privacy

OhMyShot is privacy-first:
- No analytics or tracking
- No telemetry
- No account required
- Works offline by default (Cloud Upload is optional)

If you enable Cloud Upload, your screenshot is uploaded to ImgBB to generate a shareable link.
